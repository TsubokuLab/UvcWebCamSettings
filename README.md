# UVC WebCam Settings

This is an asset for UVC(USB Video Class) Webcam.
You can use UVC WebCam with many settings.

## [ Features ]
* VideoTexture<br>
* Resolution<br>
* Framerate<br>
* Codecs<br>
* Brightness<br>
* Contrast<br>
* Saturation<br>
* Focus<br>
* Save / Load Settings<br>

This asset is using OpenCV native plugin.
Windows only.
